# TW-crawler

## プログラムの使い方

1. [Node.js](https://nodejs.org/ja/download/)をインストールする

2. コマンドプロンプトを起動する

3. コマンドをタイプする
  
    1. インストール 
    ```shell
    npm install
    ```

    2. 実行
    ```shell
    node list.js --from (FROM_DATE) --to (TO_DATE) --user (USERNAME)
    ```
    (実行結果は ```src/___.md``` ファイルに記録される)
